# AC to DC converter--PSB-Designing

An AC to DC converter PCB design project created using KiCad.

## Project Overview

This project contains the complete schematic and PCB layout design for an AC to DC power supply board. The design is production-ready and includes all necessary KiCad project files.

## Features

- Complete schematic capture of the AC to DC converter
- Fully designed PCB layout ready for manufacturing
- Project backup files to maintain version history
- Compatible with KiCad 9.0+

## Getting Started

### Prerequisites

- KiCad 9.0 or higher
- Git (for cloning the repository)

### Installation

1. **Clone the repository**
   ```bash
   git clone [https://github.com/AAISHAA1585/PSB-Designing.git](https://github.com/AAISHAA1585/PSB-Designing.git)
   ```


2. **Open the project in KiCad**
   - Launch KiCad
   - Open `Project1.kicad_pro`

3. **View or edit the schematic**
   - In KiCad, open the Schematic Editor
   - The schematic file `Project1.kicad_sch` will load automatically

4. **View or edit the PCB layout**
   - In KiCad, open the PCB Editor
   - The PCB layout file `Project1.kicad_pcb` will load automatically

## Notes

- **Do not edit** `~Project1.kicad_pcb.lck` - this file is automatically generated by KiCad when the PCB editor is open
- **Do not delete** `fp-info-cache` - this file is required by KiCad for footprint information
- Backup files are stored in the `Project1-backups/` directory for version history
